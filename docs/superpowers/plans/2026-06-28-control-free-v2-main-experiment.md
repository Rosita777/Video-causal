> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the abandoned controls-first branch with a clean, control-free v2 main experiment on the 100 human-verified clean-source prompts.

**Architecture:** V2 now has one gate before erasure: human-verified clean-source validity. The main experiment runs four erasure baselines on the verified `yes100` prompt slice and labels each erased output for target visibility and causal-footprint visibility. The deleted controls branch must not be regenerated by v2 scripts or referenced as a current requirement.

**Tech Stack:** Python scripts, pytest, CogVideoX-2B generation adapters, static HTML review pages, CSV/JSONL experiment artifacts.
